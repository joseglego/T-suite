# Usage
<!-- usage -->
```sh-session
$ npm install -g @tibia-suite/cli
$ @tibia-suite/cli COMMAND
running command...
$ @tibia-suite/cli (-v|--version|version)
@tibia-suite/cli/0.0.0 darwin-x64 node-v14.3.0
$ @tibia-suite/cli --help [COMMAND]
USAGE
  $ @tibia-suite/cli COMMAND
...
```
